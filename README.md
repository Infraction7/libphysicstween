# libphysicstween

## Getting Started
To build the model use:
```bash
argon build
```
Or install using Wally:
```bash
infraction7/physicstween@1.0.1-rc5
```

## Usage
require() the module and execute the returned function
use :Play() and :Stop() on created objects
